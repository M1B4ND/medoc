## Médecine Hospitalière
### Triage et évaluation à l’arrivée

Réalisé par l'IOA, Infirmier Organisateur d'Accueil (durée cible : 2 à 5 minutes)
   
   1. Dès que le patient arrive (brancard, marche, ambulance...)
      - Regarde l'aspect général : couleur de peau, sueurs, agitation, saignement visible, posture, respiration bruyante  ?
      - Écoute les premières plaintes ou cris. Cela te donne déjà une idée du niveau d'urgence
        
   2. Interrogatoire ultra-court
      - Qu'est-ce qui vous amène ? / Depuis quand ?
      - Avez-vous mal quelque part ? (localisation + intensité)
      - Prenez-vous des médicaments ?
      - Avez-vous des allergies ?
      - Dernier repas / événements récents ? (alcool, drogue, accident ?)

   3. Évaluation ABCDE
   4. Échelle EVDA
   5. Échelle de Glasgow
   6. Mesure des constantes vitales
        
   ### Tableau du triage
   Le niveau d’urgence est évalué dès le début et réajusté en permanence selon l’état du patient.
   | Niveau d'urgence/Priorisation | Délai max avant médecin | Signes typiques                                                                              |
   |--------------|-------------------------|----------------------------------------------------------------------------------------------|
   | Rouge        | immédiat                | arrêt cardiaque, coma Glasgow <8, hémorragie incontrôlée, détresse respiratoire extrême, ... |
   | Orange       | <10-15 minutes          | Atteinte organe vitale ou lésion sévère : douleur thoracique intense persistante, AVC <24h, TC avec otorragie/convulsions/anticoagulants, hémorragie active contrôlée, sepsis grave      |
   | Jaune        | <30 minutes             | Comorbidité liée au motif OU adressé par médecin    |
   | Vert         | <2 heures               | Urgence relative, pas de risque vital immédiat  |
   | Noir         | non urgent              | psychiatrie, décès, administratif, psychologie   |


#### Examens Complémentaires les Plus Fréquents en Urgence

| Catégorie              | Examen le plus courant                  | Quand on le demande le plus souvent                  |
|------------------------|-----------------------------------------|------------------------------------------------------|
| **Biologie sanguine**  | NFS + plaquettes                        | Toute suspicion d'infection, saignement, anémie      |
|                        | Ionogramme + urée/créatinine            | Déshydratation, choc, insuffisance rénale            |
|                        | Troponine ultrasensible                 | Douleur thoracique, suspicion SCA                    |
|                        | Lactates                                | Choc (septique, hémorragique, cardiogénique)         |
|                        | Gaz du sang artériel                    | Détresse respiratoire, acidose suspectée             | 
|                        | Glycémie                                | Malaise, coma, diabétique                            |
| **Imagerie**           | Échographie FAST                        | Polytrauma, suspicion hémopéritoine/pneumothorax     |
|                        | Scanner crânien sans injection          | Traumatisme crânien, coma, céphalée intense          |
|                        | Scanner thoraco-abdomino-pelvien (TAP)  | Polytrauma, GSW abdomen, suspicion saignement interne|
|                        | Radiographie thorax                     | Dyspnée, traumatisme thoracique, pneumonie           |
|                        | Échographie cardiaque bedside           | Choc inexpliqué, suspicion tamponnade                |
| **Fonctionnels**       | ECG 12 dérivations                      | Douleur thoracique, syncope, palpitations            |
|                        | EEG                                     | Crise convulsive prolongée, coma inexpliqué          |
|                        | Ponction lombaire                       | Suspicion méningite, hémorragie sous-arachnoïdienne  |
| **Thanatologie**       | Constat de décès (ECG + pupilles)       | Décès en service ou sur arrivée                      |
|                        | Scanner post-mortem (si mort suspecte)  | Décès inexpliqué, trauma violent                     |
